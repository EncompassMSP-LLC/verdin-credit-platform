# LRP assets

Marketing and brand binaries for Lending Readiness Partners (not application source).

```text
assets/lrp/
  logos/           # SVG/PNG wordmarks, icons, partner lockups
  canva/           # Canva export packages / source links
  presentations/   # Partner decks (PDF/PPTX)
  brochures/       # One-pagers and leave-behinds
  social/          # Social cutdowns and templates
```

Runtime web images used by Next.js live in `apps/lrp-web/public/brand/` (copy from `logos/` when shipping).
