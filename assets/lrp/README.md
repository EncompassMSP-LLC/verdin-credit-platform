# LRP assets

Marketing and brand binaries for Lending Readiness Partners (not application source).

```text
assets/lrp/
  logos/                 # SVG/PNG wordmarks, icons, partner lockups
  canva/                 # Canva export packages / source links
  presentations/         # Partner decks (PDF/PPTX)
  brochures/             # One-pagers and leave-behinds
  social/                # Social cutdowns and templates
  marketing-package/     # Turnkey outreach packs (DOCX/PPTX/XLSX/PDF)
    v2/                  # Lending_Readiness_Partners_Marketing_Package_v2
```

**v2 pack:** [`marketing-package/v2/README.md`](./marketing-package/v2/README.md) — welcome packet, partnership + borrower guides, referral forms, copy banks, pitch deck, email/social campaigns, Canva guide, outreach playbook, calendars.

Runtime web images used by Next.js live in `apps/lrp-web/public/brand/` (copy from `logos/` when shipping).

Docs index: [`docs/lrp-enterprise/`](../docs/lrp-enterprise/).
