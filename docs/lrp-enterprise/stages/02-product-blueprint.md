# Stage 2 — Product Blueprint (4–6 weeks)

**Goal:** Specify every screen before writing (or rewriting) product UI.

## Method

For each page document:

1. Purpose (one sentence)
2. Actors & permissions
3. Entry / exit routes
4. Layout regions
5. **Every field** (type, validation, source, PII class)
6. **Every button / action** (API, side effects, audit)
7. Empty, loading, error, and success states
8. Edge cases & compliance copy
9. Analytics events
