## 1. Sales motion

**Motion:** Consultative B2B — design partners first, then expand seats and borrower volume.

**Primary targets:** Credit operators; mortgage branches / LO teams; high-volume realtor teams (often via lender/operator).

## 2. Ideal opportunity criteria

Pursue when ≥3 are true:

- [ ] Mortgage-intent borrower flow exists
- [ ] Pain = status chaos / partner chasing
- [ ] Buyer rejects guarantee culture
- [ ] Ops capacity (or willingness to staff) for mediated work
- [ ] Decision-maker accessible (owner, sales manager, ops lead)

## 3. Stages

| Stage              | Exit criteria                                          |
| ------------------ | ------------------------------------------------------ |
| Prospect           | ICP fit noted                                          |
| Discovery          | Pain + volume + compliance stance confirmed            |
| Demo / walkthrough | Readiness workflow understood; non-claims acknowledged |
| Pilot design       | Success metrics + data sharing + SLA agreed            |
| Commercial         | MSA / order form (counsel)                             |
| Onboarding         | Hand off to Vol 10–12                                  |

## 4. Discovery script (core questions)

1. How do LOs/realtors get status today?
2. What does “ready to submit” mean in your world?
3. Where do deals die on credit?
4. Who owns dispute decisions?
5. What language is forbidden in your marketing/compliance?
6. Volume: referrals / month? active borrowers?
7. What would make this a must-have in 90 days?

## 5. Demo narrative (20 minutes)

| Minutes | Show                                     |
| ------- | ---------------------------------------- |
| 0–3     | Problem + non-claims                     |
| 3–8     | Tri-bureau → findings → score (advisory) |
| 8–12    | Borrower portal: tasks + learning        |
| 12–16   | Lender pipeline + export                 |
| 16–20   | Pilot plan + next steps                  |

Never demo “guaranteed outcomes.”

## 6. Objection handling

| Objection                     | Response direction                                                                       |
| ----------------------------- | ---------------------------------------------------------------------------------------- |
| “Just give us FICO”           | Readiness is multi-factor packaging for handoff; FICO remains lender’s domain            |
| “Can you auto-file disputes?” | Staff-mediated by design; unsupervised filing is out of scope                            |
| “We need LOS sync now”        | Roadmap later; prove dashboard value first                                               |
| “Price too high”              | Tie to LO hours saved + pull-through clarity; offer design-partner term                  |
| “Are you credit repair?”      | We are a readiness platform; ops may include mediated credit work with compliant process |
