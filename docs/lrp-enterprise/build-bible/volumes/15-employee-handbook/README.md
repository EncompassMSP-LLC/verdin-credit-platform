## 1. Purpose

Expectations for employees and contractors of Lending Readiness Partners (and affiliated ops under Ultimate Credit Repair / Verdin as applicable). **Not a substitute for employment counsel** — counsel finalizes before distribution.

## 2. Employment relationship (placeholders)

- At-will / contractor classification — **counsel**
- Equal opportunity
- Anti-harassment / reporting path
- Confidentiality & IP assignment

## 3. Mission & conduct

We help more borrowers become **Lending Ready** with honest, advisory language.

**Required:**

- Use claim library (Vol 18) in all borrower/partner communication
- Protect PII; least-privilege access
- Escalate compliance concerns without retaliation

**Forbidden:**

- Guaranteeing loan approval or score outcomes
- Unsupervised bureau filing
- Sharing full credit files with realtors by default
- Side deals / off-platform fee collection without approval

## 4. Employee onboarding (30-day)

### Pre-day-1

- [ ] Agreement signed
- [ ] Accounts provisioned
- [ ] Handbook + compliance ack sent

### Day-1

| Block      | Content                                     |
| ---------- | ------------------------------------------- |
| Mission    | Vision + non-claims                         |
| Product    | Borrower / lender / CRM walkthrough         |
| Compliance | Claim library; PII; staff-mediated disputes |

| Day | Focus                           |
| --- | ------------------------------- |
| 1   | Orientation + shadow            |
| 2   | Borrower portal + checklist ops |
| 3   | Findings language review        |
| 4   | Partner updates / digests       |
| 5   | Supervised solo work + retro    |

### 30-60-90

| Horizon | Expectation                          |
| ------- | ------------------------------------ |
| 30      | Independent Tier-1 work              |
| 60      | Owns a cohort / partner slice        |
| 90      | Improves one SOP or playbook section |

## 5. Work norms

| Topic        | Norm                                                    |
| ------------ | ------------------------------------------------------- |
| Hours        | Agreed with manager; partner SLAs respected             |
| Tools        | Approved systems only                                   |
| Remote       | Secure network; no PII on personal devices unsanctioned |
| Social media | No case details; brand voice if posting for company     |

## 6. Performance & feedback

- Weekly 1:1 for first 90 days
- Scorecard ties to role KPIs (Vol 06) — never to “approvals obtained”

## 7. Offboarding

1. Revoke access same day
2. Return devices / docs
3. Transfer open cases
4. Exit notes for SOP gaps

## 8. Acknowledgment

Employee signature block (counsel template): date, name, “I received and will follow this handbook and Vol 18 compliance posture.”
