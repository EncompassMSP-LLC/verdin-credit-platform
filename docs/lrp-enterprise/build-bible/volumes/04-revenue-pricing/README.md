## 1. Revenue model overview

LRP monetizes **readiness infrastructure** for operators and channels — not loan origination fees (unless a future licensed program is separately approved).

### Primary motions

| Motion            | Who pays                                                                            | What they buy                                          |
| ----------------- | ----------------------------------------------------------------------------------- | ------------------------------------------------------ |
| Operator platform | CRO / ops org                                                                       | Brand packaging, portals, CRM, analysis, partner tools |
| Lender portal     | Lender org or included in operator deal                                             | Seats, pipeline, exports                               |
| Realtor access    | Often sponsored by operator/lender                                                  | Limited visibility / referral intake                   |
| Borrower          | Usually **not** direct SaaS (B2B2C); optional consumer fee only if counsel-approved | Portal access via partner                              |

**Default GTM:** B2B2C — borrower uses portal because a partner/operator enrolled them.

## 3. Pricing principles

1. **Align to value** — partner time saved + clearer pull-through, not letter count.
2. **No guarantee pricing** — never price “approvals.”
3. **Simple enough to sell** — ≤3 tiers on the website; enterprise custom.
4. **Land then expand** — start with one lender + borrower portal; expand seats.
5. **Compliance cost is real** — don’t race to free forever plans that underfund ops.
