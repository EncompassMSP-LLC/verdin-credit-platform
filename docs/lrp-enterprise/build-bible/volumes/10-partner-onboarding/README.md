## 1. Purpose

Generic **partner** onboarding for any channel logo (lender branch, realtor team, operator affiliate). Role-specific deep dives: Vol 11 (borrower), Vol 12 (lender), Vol 13 (realtor).

## 2. Partner types

| Type               | Primary value                       | Default access        |
| ------------------ | ----------------------------------- | --------------------- |
| Lender / LO team   | Referrals + pipeline + package      | `/lender` roles       |
| Realtor team       | Coarse status + expectation setting | Limited referral view |
| Operator affiliate | Co-branded readiness ops            | CRM + portals per MSA |

## 3. Universal pre-kickoff

- [ ] ICP fit (Vol 08)
- [ ] Commercial path clear (pilot vs paid — Vol 04)
- [ ] Compliance language accepted (Vol 18)
- [ ] Named champion + backup
- [ ] Success metrics written (Vol 06)

## 4. Universal kickoff agenda (45–60 min)

1. Mission + non-claims
2. Workflow: refer → work → update → package
3. Roles and SLAs (Vol 09)
4. First-cohort plan (volume + dates)
5. Support contacts

## 5. Access provisioning

1. Create partnership record
2. Invite users with least privilege
3. Confirm login
4. Configure digests if applicable
5. Deliver Partner Kit

## 6. First-14-days gate

| Gate       | Pass criteria                       |
| ---------- | ----------------------------------- |
| Activation | ≥1 user logged in                   |
| Flow       | ≥3 referrals or equivalent activity |
| Language   | No guarantee claims observed        |
| Feedback   | Mid-pilot check-in completed        |
