## 1. Marketing job

Create **trusted demand** among operators, lenders, and realtors for a Borrower Readiness Platform — without guarantee culture or “fix your credit fast” hype.

## 2. Positioning

| Element  | Statement                                                          |
| -------- | ------------------------------------------------------------------ |
| Category | AI-powered Borrower Readiness Platform                             |
| For      | Credit operators + mortgage/realtor channels                       |
| Benefit  | Clear readiness progress partners can act on                       |
| Unlike   | Credit repair mills / consumer score apps / LOS tools              |
| Proof    | Structured analysis, advisory score, education, partner visibility |

**Tagline:** _Helping More Borrowers Become Lending Ready._

## 3. Audience messages

| Audience      | Primary message                                            | Proof points                     |
| ------------- | ---------------------------------------------------------- | -------------------------------- |
| Operators     | Productize mortgage readiness without chaos                | CRM, portals, staff-mediated ops |
| Lenders / LOs | See who’s moving toward ready — advisory, not underwriting | Pipeline, exports, digests       |
| Realtors      | Set buyer expectations with a shared signal                | Referral status, timelines       |
| Borrowers     | Know where you stand and what to do next                   | Score, plan, learning, tasks     |

## 4. Channel strategy (Year 1)

| Channel                       | Role                            | Cadence                   |
| ----------------------------- | ------------------------------- | ------------------------- |
| Founder-led partner outreach  | Design partners                 | Weekly                    |
| LinkedIn (thought leadership) | Category education              | 2–3 posts / week          |
| Partner webinars              | LO / realtor enablement         | Monthly after first logos |
| Website / SEO                 | Inbound education               | Continuous                |
| Associations / meetups        | Local beachhead                 | Quarterly                 |
| Paid ads                      | Only after claim library locked | Optional late Y1          |

**Do not** run aggressive paid “guaranteed mortgage” creatives.

## 5. Content pillars

1. Readiness ≠ credit score alone
2. Partner trust and SLA clarity
3. Compliance-minded process (staff-mediated)
4. Borrower education (utilization, collections, timelines)
5. Operator operating system (not spreadsheets)

Asset homes: `assets/lrp/social/`, `brochures/`, `presentations/`.
