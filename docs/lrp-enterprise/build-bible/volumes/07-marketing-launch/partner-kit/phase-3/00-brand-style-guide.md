# Lending Readiness Partners — Brand style guide

| Field        | Value                              |
| ------------ | ---------------------------------- |
| Status       | `ready-for-build`                  |
| Audience     | Design, marketing, sales, partners |
| Last updated | 2026-07-23                         |

Source of truth for visual and verbal identity across the Mortgage Readiness Partner Program. Claim limits: [CLAIM-LIBRARY.md](../../../CLAIM-LIBRARY.md).

---

## 1. Brand essence

| Element      | Canonical text                                                        |
| ------------ | --------------------------------------------------------------------- |
| Legal name   | Lending Readiness Partners                                            |
| Abbreviation | **LRP** only after the full name has appeared                         |
| Tagline      | Helping More Borrowers Become Lending Ready.                          |
| Supporting   | Education, strategy, and progress for the next financing opportunity. |
| Category     | AI-powered Borrower Readiness Platform / Mortgage Readiness Solutions |
| Posture      | Credible to financial professionals; understandable to consumers      |

**Not:** aggressive credit-repair marketing, guarantee culture, or “score mill” aesthetics.

---

## 2. Voice & tone

| Do                                      | Don’t                                             |
| --------------------------------------- | ------------------------------------------------- |
| Calm, precise, institutional            | Hype, urgency scams, shame language               |
| “Prepare for a lender conversation”     | “We’ll get you approved / funded”                 |
| Process, education, visibility          | Miracle timelines, fabricated before/after FICO   |
| Dignity-first borrower language         | Blame, “bad credit people,” scare tactics         |
| Partner = extension of the lending team | Substitute for underwriting or credit decisioning |

**Short disclaimer (public pieces):**

> Lending Readiness Score™ is advisory and not a loan approval or underwriting decision.

**Long disclaimer (guides, reports, landing heroes):**

> Lending Readiness Score™ is an advisory tool for organizing credit and documentation work toward a mortgage conversation. It is not a credit score from a consumer reporting agency, not an underwriting decision, and not a guarantee of loan approval or terms.

---

## 3. Color system

Print + digital. Prefer matte finishes; avoid neon “repair” gradients.

| Token        | Hex       | Use                                       |
| ------------ | --------- | ----------------------------------------- |
| Navy primary | `#0B1F3A` | Covers, headers, folder fronts            |
| Navy deep    | `#061525` | Large fields, back covers                 |
| Gold accent  | `#C4A35A` | Rules, icons, foil option, CTAs (sparing) |
| Warm paper   | `#F7F4EF` | Interior backgrounds (print)              |
| Slate text   | `#2C3340` | Body copy                                 |
| Mist         | `#E8EEF5` | Tables, callout wells                     |
| Success soft | `#2F6F4E` | Progress indicators only (not “approved”) |
| Alert soft   | `#8A4B2F` | Caution callouts (compliance)             |
| White        | `#FFFFFF` | Negative space                            |

**Avoid:** purple-to-indigo SaaS gradients, neon greens, aggressive reds for “fix your credit.”

---

## 4. Typography

| Role     | Print recommendation             | Web / Canva fallback               |
| -------- | -------------------------------- | ---------------------------------- |
| Display  | Freight Display / similar serif  | Source Serif 4 / Libre Baskerville |
| Subheads | Neue Haas Grotesk / similar sans | Source Sans 3 / Inter\*            |
| Body     | Same sans, 10–11 pt print        | 16–18 px digital                   |
| Captions | Sans, 8–9 pt                     | 12–13 px                           |

\*Inter is acceptable for product UI; marketing print should prefer a more distinctive display serif for brand moments.

**Hierarchy rule:** brand name is hero-level on covers; no competing headline that overpowers the wordmark.

---

## 5. Logo usage

| Lockup           | When to use                                                      |
| ---------------- | ---------------------------------------------------------------- |
| Primary wordmark | Covers, letterhead, folder front                                 |
| Stacked wordmark | Narrow rack cards, stories                                       |
| Icon + wordmark  | Digital headers, email                                           |
| Partner co-brand | Optional “Powered by LRP” footer only when co-marketing approved |

**Clear space:** height of the “L” in Lending around the mark.  
**Minimum width:** 1.25" print; 120 px digital.  
**Do not:** stretch, recolor outside palette, place on busy photos without scrim, pair with guarantee badges.

Logo binaries live in `assets/lrp/logos/` when delivered.

## 6. Photography & imagery

| Prefer                                              | Avoid                                       |
| --------------------------------------------------- | ------------------------------------------- |
| Homeownership atmosphere (keys, porch light, plans) | Broken credit cards, “bad credit” shame art |
| Professional desks / calm partner meetings          | Stock “celebration check” tropes            |
| Soft natural light, documentary feel                | Glowing score gauges promising FICO jumps   |
| Abstract architecture / paper texture               | AI-generated faces as “real clients”        |

**QR codes:** high-contrast black on white or navy on paper; quiet zone ≥ 4 modules; test print at final size.

---

## 7. Layout principles

1. **One composition** per cover / first viewport — not a dashboard of badges.
2. **One job per panel** — one headline, one short support line, one CTA group.
3. **Cards sparingly** — use for interactive or checklist containers only.
4. **Institutional grid** — generous margins (≥ 0.5" print); consistent footer with tagline + short disclaimer.
5. **Motion (digital):** 2–3 intentional transitions max per reel/story; no flashy particle spam.

---

## 8. Co-branding & white-label

| Mode                | Rules                                                                                            |
| ------------------- | ------------------------------------------------------------------------------------------------ |
| LRP-branded         | Full LRP identity; partner logo optional in “In partnership with”                                |
| Partner-forward     | Partner brand primary; LRP as “Mortgage Readiness Program powered by Lending Readiness Partners” |
| White-label (30.0+) | Theme + logo hooks; claim library still binds all public copy                                    |

Never allow partner marketing to add forbidden claims under LRP’s name.
