Turnkey **white-labeled Mortgage Readiness Partner Program**: sales system, print kit, borrower education, partner enablement, digital kit, and portal concepts — comparable to a national franchise leave-behind stack.

All copy is claim-library locked ([CLAIM-LIBRARY.md](../../../CLAIM-LIBRARY.md)). Binary Canva/PPT/PDF press files remain external; manuscripts and specs here are the source of truth.

## Program pillars

1. **Sales system** — folder, scripts, objections, decks, trade show
2. **Education system** — partnership guide, borrower guide, LO cheat sheet
3. **Referral system** — forms, consents, checklists, privacy
4. **Always-on marketing** — flyers, rack cards, social, email, leave-behinds
5. **Digital kit + portal** — landings, downloads, partner workspace concept
6. **Lender ops reports** — readiness / FCRA / Metro2 / progress (product-backed)

## Hard rules (every piece)

- Full name **Lending Readiness Partners** before abbreviation **LRP**
- Tagline exact: _Helping More Borrowers Become Lending Ready._
- Advisory disclaimer on public pieces
- Forbidden: guaranteed approval/funding, fabricated FICO before/after, unsupervised filing, “mortgage score” as a promise
- Success stories are **fictional composites** unless a signed testimonial release exists
