# Chapter 13 — Glossary & resources

| Term                     | Plain meaning                                              |
| ------------------------ | ---------------------------------------------------------- |
| Advisory readiness score | LRP organizer for progress—not a bureau score or approval  |
| Charge-off               | Creditor accounting write-off; debt/reporting may continue |
| Collection               | Debt placed/reported with collections                      |
| Inquiry                  | Record of a credit pull                                    |
| Overlay                  | Extra lender rule beyond base product guidelines           |
| Utilization              | Share of revolving credit in use                           |
| Underwriting             | Lender’s evaluation of the loan file                       |
