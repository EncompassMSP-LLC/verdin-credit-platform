## Front — Cheat sheet

### Header

**Lending Readiness Partners**  
Loan Officer Quick Reference

### Referral workflow

| Step | Action                                                      |
| ---- | ----------------------------------------------------------- |
| 1    | Identify near-miss / “not yet” borrower                     |
| 2    | Set expectations: advisory prep—not guaranteed approval     |
| 3    | Submit referral (`/resources/partner-kit/referral` or form) |
| 4    | Confirm borrower authorization / consent path               |
| 5    | Watch for LRP accept / needs-info acknowledgment            |
| 6    | Review weekly digest / portal status                        |
| 7    | Re-engage financing conversation when borrower is prepared  |

### Typical timelines (directional—not promises)

| File type                      | Directional range           |
| ------------------------------ | --------------------------- |
| Docs-only gaps                 | Days–weeks                  |
| Utilization / inquiry timing   | Statement cycles–months     |
| Dispute / reinvestigation work | Per bureau/statutory clocks |
| Multi-issue complex files      | Often multi-month           |

**Script line:** “Timelines depend on your file and responses from others. We’ll prioritize and keep your LO updated.”

### Common score / credit issues (education cues)

| Issue            | Borrower-facing cue (claim-safe)                    |
| ---------------- | --------------------------------------------------- |
| Utilization      | “How much of available revolving credit is in use”  |
| Collections      | “Balances reported in collections may need a plan”  |
| Charge-offs      | “Often needs documentation and a careful next step” |
| Inquiries        | “Recent applications can affect timing”             |
| Late payments    | “Payment history is heavily weighted”               |
| Thin file        | “Limited history can make decisions harder”         |
| Identity / mixed | “We verify carefully—no shame, just accuracy”       |

Do **not** quote fabricated point impacts.

### Borrower readiness checklist (hand-off)

- [ ] Government ID ready
- [ ] Proof of income path understood
- [ ] Housing history notes available
- [ ] Credit reports / portals accessible to borrower
- [ ] List of known collections / disputes
- [ ] Goals: purchase / refinance / timeline (aspirational)
- [ ] Preferred LO contact confirmed
- [ ] Consents / authorizations signed

### Partner benefits (micro)

Dedicated specialist · Intake review · Progress reports · Client portal · Rescore prep support · Claim-safe updates

### What we never claim

Guaranteed approval · Replacing underwriting · Fabricated score outcomes · Unsupervised bureau filing

## Back (optional) — Talk tracks

**When declining / delaying today**  
“You’re not out of the conversation. I’m connecting you with Lending Readiness Partners so you have a clear plan while I stay informed.”

**When asked “will this get me approved?”**  
“No one can guarantee approval. They help you prepare for the next financing conversation; I still underwrite with my lender.”

**When asked “how long?”**  
“It depends on your file. You’ll get prioritized tasks and I’ll get updates—so we’re not guessing in the dark.”
