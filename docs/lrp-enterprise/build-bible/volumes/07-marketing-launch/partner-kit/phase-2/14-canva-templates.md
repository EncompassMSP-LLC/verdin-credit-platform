# Canva template specifications

Designers: build these in Canva Brand Kit (navy / gold / sand). Copy from Phase 1 + Phase 2 docs only.

## Brand kit tokens

| Token        | Value                                        |
| ------------ | -------------------------------------------- |
| Navy         | `#00133E`                                    |
| Gold         | `#C29E5B`                                    |
| Sand / light | soft warm off-white (avoid purple gradients) |
| Type         | Display for headlines; clean sans for body   |
| Logo         | LRP wordmark; full name before LRP           |

## Mandatory on every template

- Tagline or short advisory disclaimer
- No “guaranteed approval,” fake FICO, wipe/delete claims
- Prefer homeownership / partnership imagery over “denied stamp” shock graphics
