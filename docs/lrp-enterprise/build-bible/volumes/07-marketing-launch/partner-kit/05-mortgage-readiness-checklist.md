# Mortgage readiness checklist (advisory)

Use with loan officers and borrowers as a **conversation aid**—not an underwriting decision.

Can your borrower answer **YES** to these readiness signals?

- [ ] Stable income documentation in motion
- [ ] Credit utilization managed toward healthier ranges
- [ ] No new unexplained collections
- [ ] Recent inquiries understood and limited where appropriate
- [ ] Disputed reporting items reviewed with staff (where applicable)
- [ ] Identity and personal information verified
- [ ] Mortgage-intent credit review completed (advisory)
- [ ] Down payment / gift documentation path identified
- [ ] DTI discussion started with the lender’s guidance
- [ ] Supporting documentation organized and ready to share

**Notes**

- “Lending Ready” is an LRP advisory band—not a creditor decision.
- Final eligibility always rests with the lender and applicable guidelines.

_Lending Readiness Score™ is advisory and not a loan approval or underwriting decision._
