## 1. Outcome

Realtor teams set buyer expectations with a shared readiness signal — without becoming credit counselors or underwriters.

## 2. When to onboard realtors

Prefer after at least one lender or operator pilot is stable, unless realtor is the beachhead design partner.

## 3. Kickoff (45 min)

| Time  | Topic                                     |
| ----- | ----------------------------------------- |
| 0–10  | Why readiness ≠ “pre-approved”            |
| 10–25 | Referral + status view (limited)          |
| 25–35 | Language they can use with buyers         |
| 35–45 | Who owns credit questions (operator / LO) |

## 4. Allowed realtor messaging

| Say                                    | Don’t say                           |
| -------------------------------------- | ----------------------------------- |
| “Working with a readiness program”     | “You’ll be approved”                |
| “Here’s where they are in the process” | Quote FICO as underwriting decision |
| “Ask your LO for package timing”       | Promise dispute deletions           |

## 5. Access model (draft)

| Role             | Sees                              |
| ---------------- | --------------------------------- |
| Realtor referrer | Own referrals + coarse stage      |
| Team lead        | Team referrals                    |
| No access        | Full tradelines / dispute letters |

PII minimization is mandatory.

## 6. First-14-days

- [ ] 1–2 practice referrals
- [ ] Confirm status visibility
- [ ] Align with partner LO on handoff

## 7. Exit criteria

- [ ] Referrals flowing without guarantee language
- [ ] Escalation path to LO/ops clear
