## 3. FCRA / CRO operating principles (draft)

1. Permissible purpose and consents for pulling/using reports — counsel.
2. Dispute process: accuracy investigation support is **operator-driven**, documented.
3. No deceptive marketing (CRO rules / state law — counsel).
4. Identity-theft / §605B paths: evidence reviewed by staff; no auto-submit.
5. Adverse action: LRP is not the creditor; do not issue creditor adverse-action notices as if we were.
