## 1. KPI philosophy

Measure **readiness progress and partner trust**, not vanity letter volume. Never KPI “approvals secured.”

## 2. North-star metric

**Borrowers reaching Near-ready / Mortgage-ready band per quarter** (advisory bands — not loan outcomes).

Secondary: **Partner logo net retention**.

## 5. Definitions (glossary)

| Term          | Meaning                                              |
| ------------- | ---------------------------------------------------- |
| Lending Ready | Product band / milestone — **not** a credit decision |
| Pull-through  | Referrals that reach near-ready+ within window       |
| WIP           | Borrowers with active case / program status          |

## 6. Anti-metrics (do not optimize)

- Letters mailed regardless of quality
- Score inflation to please partners
- Speed that skips compliance gates
