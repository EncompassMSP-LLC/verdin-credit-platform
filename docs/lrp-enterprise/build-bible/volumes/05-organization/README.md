## 1. Operating principle

Small senior team + clear RACI. Credit dispute execution stays with qualified ops; partners never receive unsupervised filing power.

## 3. Role definitions

| Role            | Owns                                                  | Does not own                             |
| --------------- | ----------------------------------------------------- | ---------------------------------------- |
| Founder / GM    | Strategy, capital, brand claims, partner design deals | Day-to-day every case                    |
| Compliance lead | Policies, audits, claim review, incident              | Sales quotas                             |
| Credit ops lead | Case quality, SLAs, staff-mediated filing gates       | Partner pricing                          |
| Case manager    | Borrower cases, docs, tasks, letters                  | Public marketing copy                    |
| Partner success | Lender/realtor health, digests, training              | Underwriting                             |
| LO liaison      | Referral hygiene, LO questions                        | Changing scores manually without process |
| Engineering     | Platform edition delivery from Bible                  | Inventing workflows off-spec             |

## 4. RACI (sample)

| Activity                 | Founder | Compliance | Ops | Partner success |
| ------------------------ | ------- | ---------- | --- | --------------- |
| New lender partnership   | A       | C          | C   | R               |
| Readiness export to LO   | I       | C          | A/R | C               |
| Marketing claim approval | A       | R          | I   | C               |
| Dispute send             | I       | C          | A/R | I               |
| Pricing exception        | A       | C          | I   | R               |

R = Responsible · A = Accountable · C = Consulted · I = Informed
