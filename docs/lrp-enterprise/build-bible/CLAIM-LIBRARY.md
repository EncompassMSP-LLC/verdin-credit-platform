# Claim library (canonical)

| Field        | Value                                   |
| ------------ | --------------------------------------- |
| Status       | `ready-for-build` (accepted 2026-07-22) |
| Parent       | Vol 18                                  |
| Last updated | 2026-07-22                              |

Use in all marketing, sales, CS macros, generated copy, and UI strings.

## Allowed

| Phrase / theme                                      | Notes                         |
| --------------------------------------------------- | ----------------------------- |
| Advisory readiness score / Lending Readiness Score™ | Always with disclaimer nearby |
| Partner visibility into progress                    | Status, not underwriting      |
| Staff review disputes before send                   | Staff-mediated                |
| Education and task completion                       | Habit / process               |
| Helps prepare for a lender conversation             | No outcome promise            |
| Helping More Borrowers Become Lending Ready.        | Tagline                       |

## Forbidden

| Phrase / theme                                        | Why                 |
| ----------------------------------------------------- | ------------------- |
| Guaranteed approval / pre-approval / funding          | Not a creditor      |
| We’ll get you funded / get you a mortgage             | Outcome claim       |
| AI files disputes automatically / unsupervised filing | Policy + compliance |
| Delete any negative / wipe your credit                | Deceptive CRO risk  |
| Fabricated before/after FICO                          | Misleading          |
| FCRA violation proven (without counsel)               | Legal conclusion    |

## Short UI disclaimer

> Lending Readiness Score™ is advisory and not a loan approval or underwriting decision.
