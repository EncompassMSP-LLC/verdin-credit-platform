## Brand (DOCX §02)

| Item               | Canonical text                                                                                            |
| ------------------ | --------------------------------------------------------------------------------------------------------- |
| Legal / commercial | Lending Readiness Partners                                                                                |
| Abbreviation       | **LRP** only after the full name has appeared                                                             |
| Primary tagline    | Helping More Borrowers Become Lending Ready.                                                              |
| Supporting line    | Education, strategy, and progress for the next financing opportunity.                                     |
| Brand posture      | Credible to financial professionals; understandable to consumers — not aggressive credit-repair marketing |

Avoid standalone use of “Lending” in ways that imply the company provides credit.

## Allowed

| Phrase / theme                                      | Notes                         |
| --------------------------------------------------- | ----------------------------- |
| Advisory readiness score / Lending Readiness Score™ | Always with disclaimer nearby |
| Partner visibility into progress                    | Status, not underwriting      |
| Staff review disputes before send                   | Staff-mediated                |
| Education and task completion                       | Habit / process               |
| Helps prepare for a lender conversation             | No outcome promise            |
| Helping More Borrowers Become Lending Ready.        | Tagline                       |
| Internal educational indicator of preparation       | DOCX §16 — progress, not FICO |

## Forbidden

| Phrase / theme                                                            | Why                     |
| ------------------------------------------------------------------------- | ----------------------- |
| Guaranteed approval / pre-approval / funding                              | Not a creditor          |
| We’ll get you funded / get you a mortgage                                 | Outcome claim           |
| AI files disputes automatically / unsupervised filing                     | Policy + compliance     |
| Delete any negative / wipe your credit                                    | Deceptive CRO risk      |
| Fabricated before/after FICO                                              | Misleading              |
| FCRA violation proven (without counsel)                                   | Legal conclusion        |
| FICO / mortgage score / approval likelihood / underwriting recommendation | DOCX §16 — score limits |
| AI independently makes legal conclusions or fabricates evidence           | DOCX §15                |

## Short UI disclaimer

> Lending Readiness Score™ is advisory and not a loan approval or underwriting decision.

## Longer disclaimer (reports / readiness hero)

> Lending Readiness Score™ is an advisory tool for organizing credit and documentation work toward a mortgage conversation. It is not a credit score from a consumer reporting agency, not an underwriting decision, and not a guarantee of loan approval or terms.
