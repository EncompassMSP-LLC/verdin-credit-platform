# Release Notes — Version 5.13.0 (Native Mobile Depth)

**Version:** 5.13.0  
**Date:** 2026-07-10  
**Tag:** `v5.13.0`

## Summary

Version 5.13.0 completes the **Native Mobile Depth** milestone after v5.12.0. It ships native mobile passkey client scaffold, OAuth marketplace publishing scaffold, fully autonomous bureau API filing scaffold, and unredacted cross-org benchmark export scaffold — all behind feature flags and documented **Partial** scope limits.

All four 5.13 epics ship as **Partial ✅**. See [`docs/governance/version-5.13-scope.md`](../governance/version-5.13-scope.md).

## Shipped in v5.13.0

| Area                                      | Highlights                                                                              |
| ----------------------------------------- | --------------------------------------------------------------------------------------- |
| **Native mobile passkey client**          | Operator-gated native client enrollment audit from approved passkey readiness runs      |
| **OAuth marketplace publishing**          | Admin-gated marketplace listing audit from approved OAuth developer apps                |
| **Fully autonomous bureau API filing**    | Admin-gated fully autonomous API filing audit from filed autonomous bureau filing runs  |
| **Unredacted cross-org benchmark export** | Admin-gated export audit from completed cross-org benchmark refresh runs (no live blob) |
| **Governance**                            | Capability matrix 5.13 sign-off, scope doc, completion checklist, release notes         |

## Native mobile passkey client

Admin-gated native mobile passkey client audit from `approved` mobile passkey readiness runs:

| Capability | Details                                                                                                                                                  |
| ---------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Endpoints  | `GET /enterprise/federation/native-mobile-passkey-client/status`, `GET .../runs`, `POST .../readiness-runs/{id}/start`, `POST .../runs/{run_id}/approve` |
| Audit      | `native_mobile_passkey_client_runs` table (migration `067`)                                                                                              |
| Gate       | `ENABLE_NATIVE_MOBILE_PASSKEY_CLIENT=true` (requires mobile passkey readiness)                                                                           |
| Deferred   | App store distribution, device attestation rollout, passwordless enforcement without operator gates                                                      |
