# Release Notes — Version 5.10.0 (Compliance-Gated Production Automation)

**Version:** 5.10.0  
**Date:** 2026-07-06  
**Tag:** `v5.10.0`

## Summary

Version 5.10.0 completes the **Compliance-Gated Production Automation** milestone after v5.9.0. It ships agent arbitrary execution audit, bureau re-filing scaffold, Stripe charge retry scaffold, and SAML passwordless enrollment scaffold — all behind feature flags and documented **Partial** scope limits.

All four 5.10 epics ship as **Partial ✅**. See [`docs/governance/version-5.10-scope.md`](../governance/version-5.10-scope.md).

## Shipped in v5.10.0

| Area                             | Highlights                                                                      |
| -------------------------------- | ------------------------------------------------------------------------------- |
| **Agent arbitrary execution**    | Admin-gated arbitrary execution run audit from completed unsupervised loop runs |
| **Bureau re-filing audit**       | Operator-gated re-filing run audit from filed autonomous bureau filing runs     |
| **Stripe charge retry**          | Admin-gated charge retry run audit from invoked live Stripe Tax API runs        |
| **SAML passwordless enrollment** | Federation passwordless enrollment run audit from automated SAML rotation runs  |
| **Governance**                   | Capability matrix 5.10 sign-off, scope doc, completion checklist, release notes |

## Agent arbitrary execution

Admin-gated arbitrary execution run audit from `completed` unsupervised loop runs:

| Capability | Details                                                                                                                              |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| Endpoints  | `GET /llm/arbitrary-execution/status`, `GET .../runs`, `POST .../unsupervised-runs/{id}/start`, `POST .../runs/{id}/approve`         |
| Audit      | `agent_arbitrary_execution_runs` table (migration `055`)                                                                             |
| Gate       | `ENABLE_AGENT_ARBITRARY_EXECUTION=true` (requires unsupervised loop readiness)                                                       |
| Deferred   | PII export without scrub config, fully autonomous agents without admin approval, arbitrary code execution without compliance linkage |
