# Release Notes — Version 4.3.0 GA (Operational Core)

**Version:** 4.3.0  
**Date:** 2026-06-29  
**Tag:** `v4.3.0`

## Summary

Version 4.3.0 is the **Operational Core** general availability release. It moves Verdin from platform scaffold to a cohesive credit operations product with case management, account intelligence, document processing, event-driven audit, task management, and Mission Control.

## Shipped Capabilities

- Case Management
- Credit Account Intelligence
- Document Foundation
- OCR Pipeline
- AI Classification (rules engine)
- Metadata Extraction
- Entity Resolution
- Timeline & Audit Engine
- Task Management
- Mission Control Dashboard

## Operational Core Outcome

The platform now supports an end-to-end operational workflow:

```text
Create Case
    ↓
Upload Credit Report
    ↓
OCR
    ↓
Classification
    ↓
Metadata Extraction
    ↓
Entity Resolution
    ↓
Timeline Events
    ↓
Task Management
    ↓
Mission Control Dashboard
```

## Stabilization Follow-Up

Sprint 4.3.1 intentionally pauses feature expansion to validate and harden the Operational Core:

- End-to-end workflow validation
- Performance baselines
- Security review
- Test coverage expansion

See [`docs/sprint-4.3.1/operational-core-stabilization.md`](../sprint-4.3.1/operational-core-stabilization.md).
