# Release Notes — Version 24.0.0 (Document Pipeline Recovery Parity)

**Version:** 24.0.0  
**Date:** 2026-07-21  
**Tag:** `v24.0.0`

## Summary

Version 24.0.0 advances **Compliance Intelligence Phase 23 — Document Pipeline Recovery Parity**. It closes remaining non-blocked document-pipeline recovery gaps: staff can bulk re-extract metadata for all OCR'd documents on a case (parity with case bulk credit-report re-parse), and can re-enqueue document classification when OCR exists but type was missed or stale.

Everything remains staff-driven and advisory — the platform never polls a bureau, never files a dispute, never escalates without a human, never files suit, and never shares reinvestigation-outcome data across tenants. Live bureau response ingestion execution, automated re-dispute filing execution, unsupervised escalation, automated litigation filing, and cross-tenant benchmarks remain explicitly out of scope. See [`docs/governance/version-24.0-scope.md`](../governance/version-24.0-scope.md).

## Case-level bulk metadata re-extract

| Capability | Details                                                                                             |
| ---------- | --------------------------------------------------------------------------------------------------- |
| Surface    | Case Detail → Credit report history → Re-extract metadata                                           |
| API        | `POST /cases/{id}/metadata/reextract`                                                               |
| Guardrail  | Soft-skip missing OCR / enqueue failure; 503 when metadata disabled; queued/skipped counts; no auto |

## Operator re-classify document

| Capability | Details                                                                                    |
| ---------- | ------------------------------------------------------------------------------------------ |
| Surface    | Document Detail → Re-classify document (when OCR present)                                  |
| API        | `POST /documents/{id}/classify/reclassify`                                                 |
| Guardrail  | Org-scoped; 422 without OCR; 503 when classification disabled; sync classify endpoint kept |
