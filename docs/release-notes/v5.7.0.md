# Release Notes — Version 5.7.0 (Compliance-Gated Autonomous Workflows)

**Version:** 5.7.0  
**Date:** 2026-07-05  
**Tag:** `v5.7.0`

## Summary

Version 5.7.0 completes the **Compliance-Gated Autonomous Workflows** milestone after v5.6.0. It ships dispute bureau submission scaffold, agent external tool-calling audit, SAML certificate rotation scaffold, and Stripe invoice PDF generation audit — all behind feature flags and documented **Partial** scope limits.

All four 5.7 epics ship as **Partial ✅**. See [`docs/governance/version-5.7-scope.md`](../governance/version-5.7-scope.md).

## Shipped in v5.7.0

| Area                   | Highlights                                                                       |
| ---------------------- | -------------------------------------------------------------------------------- |
| **Dispute bureau**     | Admin-gated bureau submission run audit linked to prepared filing prep runs      |
| **Agent tool-calling** | Human-gated external tool invocation audit with admin approval                   |
| **SAML cert rotation** | Federation certificate rotation run audit linked to valid SAML metadata uploads  |
| **Stripe invoice PDF** | Admin-gated invoice PDF generation run audit linked to completed collection runs |
| **Governance**         | Capability matrix 5.7 sign-off, scope doc, completion checklist, release notes   |

## Dispute bureau submission

Staff-submitted bureau submission runs requiring admin approval before any external filing scaffold:

| Capability | Details                                                                                                                            |
| ---------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| Endpoints  | `GET /compliance/dispute-bureau-submission/status`, `GET .../runs`, `POST .../prep-runs/{id}/submit`, `POST .../runs/{id}/approve` |
| Audit      | `dispute_bureau_submission_runs` table (migration `043`)                                                                           |
| Timeline   | Approvals emit `dispute_bureau_submission` timeline events on linked cases                                                         |
| Gate       | `ENABLE_DISPUTE_BUREAU_SUBMISSION=true` (requires `prepared` filing prep run)                                                      |
| Deferred   | Unsupervised filing loops, live bureau API integration without compliance deferral docs                                            |

## Agent external tool-calling

Human-gated external tool invocation audit with admin approval and optional case timeline correlation:

| Capability | Details                                                                                                   |
| ---------- | --------------------------------------------------------------------------------------------------------- |
| Endpoints  | `GET /llm/tool-calling/status`, `GET .../requests`, `POST .../requests`, `POST .../requests/{id}/approve` |
| Audit      | `agent_tool_invocation_requests` table (migration `044`)                                                  |
| Gate       | `ENABLE_AGENT_EXTERNAL_TOOL_CALLING=true` (requires agent execution readiness)                            |
| Deferred   | Unsupervised tool loops, arbitrary code execution, PII export without scrub config                        |

## SAML certificate rotation

Admin-gated federation certificate rotation run audit linked to valid SAML metadata uploads:

| Capability | Details                                                                                                                                       |
| ---------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| Endpoints  | `GET /enterprise/federation/saml-cert-rotation/status`, `GET .../runs`, `POST .../metadata-uploads/{id}/rotate`, `POST .../runs/{id}/approve` |
| Audit      | `saml_certificate_rotation_runs` table (migration `045`)                                                                                      |
| Gate       | `ENABLE_SAML_CERTIFICATE_ROTATION=true` (requires HRIS sync + SAML metadata readiness)                                                        |
| Deferred   | Automated rotation without operator review, multi-IdP bulk rotation UI                                                                        |

## Stripe invoice PDF

Admin-gated invoice PDF generation run audit linked to completed `invoice_pdf` collection runs:

| Capability | Details                                                                                                                   |
| ---------- | ------------------------------------------------------------------------------------------------------------------------- |
| Endpoints  | `GET /billing/invoice-pdf/status`, `GET .../runs`, `POST .../collection-runs/{id}/generate`, `POST .../runs/{id}/approve` |
| Audit      | `stripe_invoice_pdf_runs` table (migration `046`)                                                                         |
| Gate       | `ENABLE_STRIPE_INVOICE_PDF=true` (requires `ENABLE_BILLING_INVOICE_COLLECTION`)                                           |
| Deferred   | Live Stripe invoice PDF API calls, tax calculation, automated charge retries                                              |

## Feature flags

| Flag                                 | Purpose                                      |
| ------------------------------------ | -------------------------------------------- |
| `ENABLE_DISPUTE_BUREAU_SUBMISSION`   | Bureau submission run audit scaffold         |
| `ENABLE_AGENT_EXTERNAL_TOOL_CALLING` | External tool invocation audit scaffold      |
| `ENABLE_SAML_CERTIFICATE_ROTATION`   | SAML certificate rotation run audit scaffold |
| `ENABLE_STRIPE_INVOICE_PDF`          | Stripe invoice PDF generation audit scaffold |
| `ENABLE_DISPUTE_FILING_PREP`         | Required for bureau submission readiness     |
| `ENABLE_AGENT_EXECUTION`             | Required for tool-calling readiness          |
| `ENABLE_HRIS_BIDIRECTIONAL_SYNC`     | Required for SAML cert rotation readiness    |
| `ENABLE_BILLING_INVOICE_COLLECTION`  | Required for Stripe invoice PDF readiness    |
