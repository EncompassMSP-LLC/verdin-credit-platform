# Release Notes — Version 5.9.0 (Compliance-Gated Autonomous Production)

**Version:** 5.9.0  
**Date:** 2026-07-05  
**Tag:** `v5.9.0`

## Summary

Version 5.9.0 completes the **Compliance-Gated Autonomous Production** milestone after v5.8.0. It ships agent unsupervised loop audit, autonomous bureau filing scaffold, live Stripe Tax API invocation audit, and SAML automated rotation scaffold — all behind feature flags and documented **Partial** scope limits.

All four 5.9 epics ship as **Partial ✅**. See [`docs/governance/version-5.9-scope.md`](../governance/version-5.9-scope.md).

## Shipped in v5.9.0

| Area                         | Highlights                                                                     |
| ---------------------------- | ------------------------------------------------------------------------------ |
| **Agent unsupervised loops** | Multi-step loop audit without per-step human gates; admin approval to complete |
| **Autonomous bureau filing** | Operator-gated autonomous filing run audit from invoked bureau live API runs   |
| **Live Stripe Tax API**      | Admin-gated live Stripe Tax calculation invocation audit scaffold              |
| **SAML automated rotation**  | Federation cert automated rotation run audit from rotated cert rotation runs   |
| **Governance**               | Capability matrix 5.9 sign-off, scope doc, completion checklist, release notes |

## Agent unsupervised loops

Multi-step agent loop audit without per-step human gates, requiring a `completed` supervised loop run before start:

| Capability | Details                                                                                                                   |
| ---------- | ------------------------------------------------------------------------------------------------------------------------- |
| Endpoints  | `GET /llm/unsupervised-loops/status`, `GET .../runs`, `POST .../supervised-runs/{id}/start`, `POST .../runs/{id}/approve` |
| Audit      | `agent_unsupervised_loop_runs` table (migration `051`)                                                                    |
| Gate       | `ENABLE_AGENT_UNSUPERVISED_LOOPS=true` (requires supervised loop readiness)                                               |
| Deferred   | Arbitrary code execution, PII export without scrub config, loops without compliance policy linkage                        |

## Autonomous bureau filing

Operator-gated autonomous filing run audit from `invoked` bureau live API runs:

| Capability | Details                                                                                                                             |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| Endpoints  | `GET /compliance/autonomous-bureau-filing/status`, `GET .../runs`, `POST .../live-api-runs/{id}/file`, `POST .../runs/{id}/approve` |
| Audit      | `autonomous_bureau_filing_runs` table (migration `052`)                                                                             |
| Gate       | `ENABLE_AUTONOMOUS_BUREAU_FILING=true` (requires bureau live API readiness)                                                         |
| Deferred   | Unsupervised re-filing loops, filing without operator review, automated re-dispute scheduling                                       |

## Live Stripe Tax API

Admin-gated live Stripe Tax calculation invocation audit from `calculated` tax calculation runs:

| Capability | Details                                                                                                                       |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------- |
| Endpoints  | `GET /billing/live-tax-api/status`, `GET .../runs`, `POST .../tax-calculation-runs/{id}/invoke`, `POST .../runs/{id}/approve` |
| Audit      | `stripe_live_tax_api_runs` table (migration `053`)                                                                            |
| Gate       | `ENABLE_STRIPE_LIVE_TAX_API=true` (requires tax calculation readiness)                                                        |
| Deferred   | Live Stripe Tax API calls without compliance deferral docs, automated charge retries, dunning legal template generation       |
