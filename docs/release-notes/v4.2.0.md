# Release Notes

## v4.2.0 — Sprint 1: Platform Foundation

**Release Date:** 2026-06-28

### Added

- Monorepo with pnpm workspaces and Turborepo
- FastAPI backend with layered architecture (router → service → repository)
- React frontend with Vite, TypeScript, and Tailwind CSS
- PostgreSQL database with SQLAlchemy 2.x async ORM
- Alembic migrations with initial schema
- JWT authentication with refresh tokens
- Role-based access control (Owner, Admin, Case Manager, Reviewer, Read Only)
- Database models: User, Organization, Case, Account, Document, Task, Communication, TimelineEvent
- Docker Compose stack (API, Web, Worker, PostgreSQL, Redis, MinIO, Nginx)
- GitHub Actions CI pipeline (lint, typecheck, tests, build, docker)
- Shared packages: `@verdin/shared`, `@verdin/ui`, `@verdin/validation`
- Engineering documentation and AGENTS.md
- Cursor rules for AI-assisted development
- Database seed script with demo data

### API Endpoints

- `GET /api/v1/health` — Health check
- `GET /api/v1/version` — Version info
- `POST /api/v1/auth/login` — User login
- `POST /api/v1/auth/refresh` — Token refresh
- `GET /api/v1/auth/me` — Current user

### Frontend Pages

- Dashboard, Cases, Accounts, Documents, Tasks, Settings
- Login page with form validation
- Protected route authentication

### Infrastructure

- Nginx reverse proxy configuration
- PostgreSQL initialization with extensions
- Background worker scaffold
