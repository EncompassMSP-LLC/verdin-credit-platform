# Release Notes — Version 5.14.0 (Production Distribution Depth)

**Version:** 5.14.0  
**Date:** 2026-07-10  
**Tag:** `v5.14.0`

## Summary

Version 5.14.0 completes the **Production Distribution Depth** milestone after v5.13.0. It ships live unredacted benchmark blob export scaffold, unsupervised autonomous filing loop audit, public OAuth marketplace listing scaffold, and native mobile app store distribution readiness — all behind feature flags and documented **Partial** scope limits.

All four 5.14 epics ship as **Partial ✅**. See [`docs/governance/version-5.14-scope.md`](../governance/version-5.14-scope.md).

## Shipped in v5.14.0

| Area                                      | Highlights                                                                                          |
| ----------------------------------------- | --------------------------------------------------------------------------------------------------- |
| **Live unredacted benchmark blob export** | Admin-gated placeholder blob export from approved unredacted export runs (secure storage reference) |
| **Unsupervised autonomous filing loops**  | Operator-gated filing loop audit from executed fully autonomous bureau API filing runs              |
| **Public OAuth marketplace listings**     | Admin-gated public listing audit from approved marketplace publishing runs                          |
| **Native mobile app store distribution**  | Admin-gated app store distribution readiness from approved native passkey client runs               |
| **Governance**                            | Capability matrix 5.14 sign-off, scope doc, completion checklist, release notes                     |

## Live unredacted benchmark blob export

Admin-gated live blob export pipeline scaffold from `approved` unredacted cross-org benchmark export runs:

| Capability | Details                                                                                                                                                    |
| ---------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Endpoints  | `GET /reporting/live-unredacted-benchmark-blob-exports/status`, `GET .../runs`, `POST .../unredacted-export-runs/{id}/start`, `POST .../runs/{id}/approve` |
| Audit      | `live_unredacted_benchmark_blob_export_runs` table (migration `077`)                                                                                       |
| Gate       | `ENABLE_LIVE_UNREDACTED_BENCHMARK_BLOB_EXPORT=true` (requires unredacted export readiness)                                                                 |
| Deferred   | Unrestricted cross-tenant PII dumps, public download links without auth                                                                                    |
