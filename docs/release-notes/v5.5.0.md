# Release Notes — Version 5.5.0 (Production Automation)

**Version:** 5.5.0  
**Date:** 2026-07-04  
**Tag:** `v5.5.0`

## Summary

Version 5.5.0 completes the **Production Automation** milestone after v5.4.0. It ships billing invoice collection, SAML federation metadata upload, marketing SMS delivery worker, and human-gated agent execution — all behind feature flags and documented **Partial** scope limits.

All four 5.5 epics ship as **Partial ✅**. See [`docs/governance/version-5.5-scope.md`](../governance/version-5.5-scope.md).

## Shipped in v5.5.0

| Area                       | Highlights                                                                                                  |
| -------------------------- | ----------------------------------------------------------------------------------------------------------- |
| **Billing collection**     | Invoice PDF + payment collection reminder scaffold behind `ENABLE_BILLING_INVOICE_COLLECTION`               |
| **SAML metadata**          | Federation metadata upload and validation audit behind `ENABLE_SAML_FEDERATION_METADATA`                    |
| **Marketing SMS delivery** | `sms_marketing_campaign_delivery` worker job + Twilio delivery audit behind `ENABLE_SMS_MARKETING_DELIVERY` |
| **Agent execution**        | Human-gated step submit/approve audit + timeline correlation behind `ENABLE_AGENT_EXECUTION`                |
| **Governance**             | Capability matrix 5.5 sign-off, scope doc, completion checklist, release notes                              |

## Billing invoice collection

Org-scoped invoice PDF and payment collection reminder scaffold:

| Capability | Details                                                                                          |
| ---------- | ------------------------------------------------------------------------------------------------ |
| Endpoints  | `GET /billing/collection/status`, `GET /billing/collection/runs`, `POST /billing/collection/run` |
| Audit      | `billing_invoice_collection_runs` table (migration `036`)                                        |
| Gate       | `ENABLE_BILLING_INVOICE_COLLECTION=true` (requires `ENABLE_BILLING_INVOICING`)                   |
| Deferred   | Stripe invoice PDF API calls, tax calculation, automated charge retries                          |

## SAML federation metadata

Per-tenant SAML metadata upload aligned with IdP federation registry:

| Capability | Details                                                                                 |
| ---------- | --------------------------------------------------------------------------------------- |
| Endpoints  | `GET /enterprise/federation/saml-metadata/status`, `GET .../uploads`, `POST .../upload` |
| Audit      | `saml_federation_metadata_uploads` table (migration `037`)                              |
| Gate       | `ENABLE_SAML_FEDERATION_METADATA=true` (requires `ENABLE_IDP_FEDERATION`)               |
| Deferred   | HRIS bidirectional sync, passwordless enrollment UI, automated certificate rotation     |

## Marketing SMS delivery

Worker-driven delivery for queued marketing SMS campaign runs:

| Capability | Details                                                                                                                   |
| ---------- | ------------------------------------------------------------------------------------------------------------------------- |
| Endpoints  | `GET /notifications/sms-campaigns/status`, `GET .../runs`, `POST .../run` (pending run + worker enqueue when delivery on) |
| Worker     | `sms_marketing_campaign_delivery` job; delivery audit via `sms_delivery_logs.campaign_run_id` (migration `038`)           |
| Gate       | `ENABLE_SMS_MARKETING_DELIVERY=true` (requires `ENABLE_SMS_MARKETING_CAMPAIGNS` + Twilio config)                          |
| Deferred   | Deliverability dashboards, multi-provider failover, A/B testing, opt-out compliance automation                            |

## Agent execution (human-gated)

Staff-submitted agent steps requiring admin approval before execution scaffold is recorded:

| Capability | Details                                                                                       |
| ---------- | --------------------------------------------------------------------------------------------- |
| Endpoints  | `GET /llm/execution/status`, `GET /llm/execution/steps`, `POST .../steps`, `POST .../approve` |
| Audit      | `agent_execution_steps` table (migration `039`)                                               |
| Timeline   | Case-scoped approvals emit `agent_execution_step` timeline events                             |
| Gate       | `ENABLE_AGENT_EXECUTION=true` (requires `ENABLE_AGENT_OBSERVABILITY`)                         |
| Deferred   | Autonomous dispute filing, external tool calling without approval, unsupervised agent loops   |

## Feature flags

| Flag                                | Purpose                               |
| ----------------------------------- | ------------------------------------- |
| `ENABLE_BILLING_INVOICE_COLLECTION` | Invoice collection run scaffold       |
| `ENABLE_SAML_FEDERATION_METADATA`   | SAML metadata upload + validation     |
| `ENABLE_SMS_MARKETING_DELIVERY`     | Marketing SMS worker delivery + audit |
| `ENABLE_AGENT_EXECUTION`            | Human-gated agent step submit/approve |
| `ENABLE_BILLING_INVOICING`          | Required for invoice collection       |
| `ENABLE_IDP_FEDERATION`             | Required for SAML metadata            |
| `ENABLE_SMS_MARKETING_CAMPAIGNS`    | Required for marketing SMS delivery   |
| `ENABLE_AGENT_OBSERVABILITY`        | Required for agent execution          |
