# Release Notes — Version 25.0.0 (Document Pipeline Recovery Bulk Closeout)

**Version:** 25.0.0  
**Date:** 2026-07-21  
**Tag:** `v25.0.0`

## Summary

Version 25.0.0 advances **Compliance Intelligence Phase 24 — Document Pipeline Recovery Bulk Closeout**. It closes remaining non-blocked **case-scoped bulk** document-pipeline recovery gaps: staff can bulk re-classify OCR'd documents on a case, and can bulk retry OCR for failed eligible documents from Credit Report History.

Everything remains staff-driven and advisory — the platform never polls a bureau, never files a dispute, never escalates without a human, never files suit, and never shares reinvestigation-outcome data across tenants. Live bureau response ingestion execution, automated re-dispute filing execution, unsupervised escalation, automated litigation filing, and cross-tenant benchmarks remain explicitly out of scope. See [`docs/governance/version-25.0-scope.md`](../governance/version-25.0-scope.md).

## Case-level bulk re-classify

| Capability | Details                                                                                          |
| ---------- | ------------------------------------------------------------------------------------------------ |
| Surface    | Case Detail → Credit report history → Re-classify documents                                      |
| API        | `POST /cases/{id}/classify/reclassify`                                                           |
| Guardrail  | Soft-skip missing OCR / enqueue failure; 503 when classification disabled; queued/skipped counts |

## Case-level bulk OCR retry (failed docs)

| Capability | Details                                                                                                 |
| ---------- | ------------------------------------------------------------------------------------------------------- |
| Surface    | Case Detail → Credit report history → Retry failed OCR                                                  |
| API        | `POST /cases/{id}/ocr/retry`                                                                            |
| Guardrail  | Soft-skip not failed / not OCR-eligible / enqueue failure; 503 when OCR disabled; queued/skipped counts |
