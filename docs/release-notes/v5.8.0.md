# Release Notes — Version 5.8.0 (Compliance-Gated Production Integrations)

**Version:** 5.8.0  
**Date:** 2026-07-05  
**Tag:** `v5.8.0`

## Summary

Version 5.8.0 completes the **Compliance-Gated Production Integrations** milestone after v5.7.0. It ships agent supervised loop audit, bureau live API invocation scaffold, Stripe tax calculation audit, and HRIS lifecycle sync scaffold — all behind feature flags and documented **Partial** scope limits.

All four 5.8 epics ship as **Partial ✅**. See [`docs/governance/version-5.8-scope.md`](../governance/version-5.8-scope.md).

## Shipped in v5.8.0

| Area                       | Highlights                                                                     |
| -------------------------- | ------------------------------------------------------------------------------ |
| **Agent supervised loops** | Multi-step loop audit with human gates between invoked tool-calling steps      |
| **Bureau live API**        | Operator-gated external bureau API invocation audit from submitted runs        |
| **Stripe tax calculation** | Admin-gated tax calculation run audit from generated invoice PDF runs          |
| **HRIS lifecycle sync**    | Admin-gated employee lifecycle sync audit from completed bidirectional runs    |
| **Governance**             | Capability matrix 5.8 sign-off, scope doc, completion checklist, release notes |

## Agent supervised loops

Human-gated multi-step agent loop audit requiring an `invoked` tool invocation request before each supervised step:

| Capability | Details                                                                                                               |
| ---------- | --------------------------------------------------------------------------------------------------------------------- |
| Endpoints  | `GET /llm/supervised-loops/status`, `GET .../runs`, `POST .../tool-requests/{id}/start`, `POST .../runs/{id}/approve` |
| Audit      | `agent_supervised_loop_runs` table (migration `047`)                                                                  |
| Timeline   | Approvals emit `agent_supervised_loop` timeline events on linked cases when present                                   |
| Gate       | `ENABLE_AGENT_SUPERVISED_LOOPS=true` (requires tool-calling readiness)                                                |
| Deferred   | Fully unsupervised loops, arbitrary code execution, PII export without scrub config                                   |

## Bureau live API integration

Operator-gated external bureau API invocation audit from `submitted` bureau submission runs:

| Capability | Details                                                                                                                        |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------ |
| Endpoints  | `GET /compliance/bureau-live-api/status`, `GET .../runs`, `POST .../submission-runs/{id}/invoke`, `POST .../runs/{id}/approve` |
| Audit      | `bureau_live_api_runs` table (migration `048`)                                                                                 |
| Timeline   | Approvals emit `bureau_live_api_invocation` timeline events on linked cases                                                    |
| Gate       | `ENABLE_BUREAU_LIVE_API=true` (requires bureau submission readiness)                                                           |
| Deferred   | Unsupervised filing loops, live bureau calls without operator review, automated re-dispute scheduling                          |
